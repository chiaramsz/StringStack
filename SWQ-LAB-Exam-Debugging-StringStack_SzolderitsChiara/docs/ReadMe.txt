SWQ LAB Exam: StringStack Debugging
==========================================
SWQ-LAB-Exam ist eine simple Stack Implementierung mit folgenden Funktionalitäten:
push()
pop()

1.Abschnitt:
------------
Benennen Sie das Projekt nach folgendem Schema um: "SWQ-LAB-Exam-Debugging-StringStack_Name1Name2"
(Ersetzen Sie "NameX" durch Ihre Namen.)

Stellen Sie das Projekt unter Versionskontrolle (git!). Checken Sie regelmäßig zu sinnvollen Zeitpunkten ein.

2.Abschnitt:
------------
Konfigurieren Sie Ihre Java Projekt so, dass Sie mit Hilfe von JUNIT testen
können. (Hinweis: Gradle bzw. Projekteigenschaften)

3.Abschnitt:
------------
Bauen Sie in Ihre Klassen Logger und SINNVOLLE Log statements ein

4.Abschnitt:
------------
Korrigieren Sie die Tests, und eventuell die Implementierung, sodass alle Tests "grün" sind.
Die Tests dürfen nicht so verändert werden, dass die überprüfte Funktionalität eine andere ist. 

Abgabe:
----------------
Zippen sie den gesamten Ordner (inc. .git directory) 
Viel Spaß und gutes Gelingen!
